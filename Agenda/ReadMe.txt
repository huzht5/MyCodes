在控制终端里进入到agenda/目录下，即可输入命令
用户注册 go run main.go register -u=[username] -p=[password] -e=[email] -n=[phoneNumber]
用户登录 go run main.go login -u=[username] -p=[password]
用户登出 go run main.go logout
用户查询 go run main.go query
用户删除 go run main.go delete
上述命令里的[]中的参数是用户输入的参数 username用户名 password用户密码 email邮箱地址 phoneNumber手机号码
